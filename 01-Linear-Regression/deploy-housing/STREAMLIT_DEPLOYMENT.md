## ðŸ› Troubleshooting

### "Model not found" Error
Run `python save_model.py` first to generate `housing_model.pkl`

### Streamlit Not Found
```bash
pip install streamlit
```

### Port Already in Use
```bash
streamlit run app.py --server.port 8502
```

### Deployment Fails on Cloud
- Ensure `requirements.txt` is in the same folder as `app.py`
- Check that `housing_model.pkl` is committed to GitHub
